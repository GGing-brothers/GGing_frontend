## 앱 기능
사용자의 동의하에 자신의 운동영상을 공유->상급자들의 피드백 (커뮤니티)

mediapipe에서 잘못된 자세가 나오면 앱에서 피드백 (비대칭 자세 확인, 올바른 자세)

근전도 2개 왼쪽, 오른쪽 (양쪽의 근육 사용량 체크->비대칭 방지)

하드웨어 사용법 안내

운동 일지 기록

운동 세트와 개수는 사용자가 설정

개수를 운동 시작하면 앱에다 표시

운동 자세 안내 [그림, 사진, 동영상 고민을 해야함]
